Macro Definition Documentation

#define LOG_ORIGIN   __FILE__

default origin.

You may (pre)define LOG_ORIGIN to a string that describes the log message source (e.g. the module). It's also possible to define this to NULL. If you don't pass a value yourself, it will default to the current filename.

1 #define LOG_ORIGIN "foobar"
2 #include "log.h"
