Detailed Description

binary logging.

This file implements logging backends that output to log files in 'native' MessagePack format. The resulting file is a binary stream of msgpack objects preserving all log information in a more compact format (compared to a textual representation). The file contents can be read back by calling e.g. msgpack_unpacker_next() in a loop.

Use log_binary() to register a new logger.
