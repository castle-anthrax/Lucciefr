#define TRY   do

start a TRY block.

TRY { ... } FINALLY is syntactic sugar for creating an explicit do { ... } while (0); block. The idea is that such a block can be left with a break statement in case of errors, to properly handle cleaning up etc. 

stuff = malloc(some_size);
TRY {
    do_something_with_stuff();
    if (error) break;
    do_more_stuff();
} FINALLY
free(stuff);
